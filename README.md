# Frontend Interview Questions

Интерактивное приложение для подготовки к собеседованиям по Frontend разработке.

## Возможности

- **1085+ вопросов** по 11 категориям
- **Отслеживание прогресса** - отмечайте изученные вопросы
- **Темная и светлая темы**
- **Адаптивный дизайн** для мобильных и десктопов
- **Локальное хранение** прогресса

## Технологии

- React 19
- TypeScript
- Ant Design
- Vite

## Категории

- JavaScript (344 вопроса)
- React (161)
- TypeScript (44)
- CSS (89)
- Общее (265)
- Асинхронность (108)
- Процессы (33)
- Менеджеры состояния (19)
- Архитектура (5)
- Git (12)
- Тесты (5)
